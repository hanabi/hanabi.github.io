- All 3's have been played on the stacks.
- Alice clues number 3 to Cathy, touching one card on slot 3.
- What type of clue is this?
- What notes should be written?

<Question />

<TabItem value="solution">

- No more 3's should be played on the stacks, so the touched card is known trash.
- This is a _Trash Chop Move_, which _Chop Moves_ all cards to the right of the trash.
- All players should write `kt` on Bob's slot 3 and `cm` on Bob's slot 4 and slot 5.
